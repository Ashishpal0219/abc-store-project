## ‚ú® Key Features
* [cite_start]**Hero Slider:** Auto-sliding banner showcasing offers and featured products[cite: 21, 22].
* [cite_start]**Dynamic Product Listing:** All products are fetched in real-time from the MongoDB database[cite: 25, 26].
* **Category Filtering:** Users can filter products (e.g., Staples, Vegetables, Snacks).
* [cite_start]**Shopping Cart:** Fully functional cart with Add/Remove functionality and dynamic total calculation[cite: 33, 34].
* [cite_start]**Mock Payment Gateway:** A realistic payment popup that simulates transaction processing (Success/Failure states)[cite: 40].
* **Custom Notifications:** "Toast" style popup notifications when items are added to the cart.
* [cite_start]**Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop screens[cite: 20].

## ‚öôÔ∏è Installation & Run Locally

If you want to run this project on your local machine, follow these steps:

1. Clone the Repository
```bash
git clone [https://github.com/Ashishpal0219/abc-store-project.git](https://github.com/Ashishpal0219/abc-store-project.git)
cd abc-store-

2. Backend Setup
Open a terminal and navigate to the server folder:

Bash

cd server
npm install
# Note: Ensure you have MongoDB installed locally or a .env file with your MONGO_URI
node index.js
The backend server will start on port 5000.

3. Frontend Setup
Open a new terminal window and navigate to the client folder:

Bash

cd client
npm install
npm run dev
The frontend will start on http://localhost:5173 (or similar).
