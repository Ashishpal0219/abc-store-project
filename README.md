## ‚ú® Key Features
* **Hero Slider:** Auto-sliding banner showcasing offers and featured products.
* **Dynamic Product Listing:** All products are fetched in real-time from the MongoDB database.
* **Category Filtering:** Users can filter products (e.g., Staples, Vegetables, Snacks).
* **Shopping Cart:** Fully functional cart with Add/Remove functionality and dynamic total calculation.
* **Mock Payment Gateway:** A realistic payment popup that simulates transaction processing (Success/Failure states).
* **Custom Notifications:** "Toast" style popup notifications when items are added to the cart.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop screens.

To run this project on your local machine, follow these steps:

### 1. Clone the Repository
```bash

3. Frontend Setup
Open a new terminal window and navigate to the client folder:

Bash
